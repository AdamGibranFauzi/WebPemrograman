Data Preprocessing

Pengertian Data Preprocessing

Data preprocessing adalah tahapan awal dalam Machine Learning yang bertujuan untuk membersihkan, menyiapkan, dan mentransformasikan data agar siap digunakan dalam proses pelatihan model.

Kualitas data sangat berpengaruh terhadap performa model Machine Learning.

Tahapan Data Preprocessing

  • Data Cleaning (pembersihan data)
  • Handling Missing Values
  • Data Transformation
  • Data Normalization & Standardization
  • Feature Selection

Data Cleaning

Data cleaning bertujuan untuk menghilangkan kesalahan pada data, seperti data duplikat, inkonsistensi format, dan nilai yang tidak valid.

  • Menghapus data duplikat
  • Memperbaiki kesalahan penulisan
  • Menghapus data yang tidak relevan

Handling Missing Values

Missing value adalah kondisi di mana data tidak memiliki nilai. Penanganannya dapat dilakukan dengan beberapa cara:

  • Menghapus baris data yang kosong
  • Mengisi dengan nilai rata-rata (mean)
  • Mengisi dengan nilai tengah (median)
  • Mengisi dengan nilai yang paling sering muncul (mode)

Normalisasi dan Standardisasi

Normalisasi dan standardisasi digunakan untuk menyamakan skala data sehingga model dapat bekerja lebih optimal.

  • Normalisasi → Mengubah data ke rentang 0–1
  • Standardisasi → Mengubah data ke distribusi normal

Contoh Data Preprocessing dengan Python

import pandas as pd
from sklearn.preprocessing import MinMaxScaler

data = pd.read_csv("data.csv")

# Mengisi missing value
data.fillna(data.mean(), inplace=True)

# Normalisasi data
scaler = MinMaxScaler()
data_scaled = scaler.fit_transform(data)
