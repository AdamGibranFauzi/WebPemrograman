Pengenalan Machine Learning dan Instalasi Python

Pengertian Machine Learning

Machine Learning adalah cabang dari Artificial Intelligence (AI) yang memungkinkan sistem komputer belajar dari data dan meningkatkan performanya tanpa diprogram secara eksplisit.

Dengan Machine Learning, sistem dapat mengenali pola, membuat prediksi, dan mengambil keputusan berdasarkan data.


Perbedaan AI, Machine Learning, dan Deep Learning

  • Artificial Intelligence (AI) → Konsep umum kecerdasan buatan
  • Machine Learning (ML) → Bagian dari AI yang belajar dari data
  • Deep Learning (DL) → Bagian dari ML yang menggunakan neural network

Contoh Penerapan Machine Learning

  • Sistem rekomendasi (YouTube, Netflix)
  • Prediksi kelulusan mahasiswa
  • Deteksi spam email
  • Pengenalan wajah
  • Prediksi harga

Cara Instalasi Python (Windows)

  1. Buka website resmi Python: https://www.python.org
  2. Klik menu Downloads
  3. Pilih Python 3.x (Latest Version)
  4. Download file installer
  5. Jalankan installer dan centang opsi "Add Python to PATH"
  6. Klik Install Now
  7. Tunggu hingga proses instalasi selesai

Cek Instalasi Python

Setelah instalasi selesai, buka Command Prompt (CMD) lalu ketik perintah berikut: 

python --version

Jika Python berhasil terinstal, maka akan muncul versi Python yang terpasang di komputer.
