Object Segmentation

Pengenalan Object Segmentation

Object Segmentation adalah teknik dalam Computer Vision yang bertujuan untuk memisahkan setiap piksel gambar ke dalam kelas tertentu sehingga objek dapat diidentifikasi secara lebih detail dibandingkan object detection.

Perbedaan Detection dan Segmentation

  • Object Detection: Menggunakan bounding box
  • Object Segmentation: Menggunakan mask piksel

Jenis Object Segmentation

  • Semantic Segmentation
  • Instance Segmentation
  • Panoptic Segmentation

Studi Kasus Object Segmentation

Object segmentation digunakan dalam berbagai bidang, seperti segmentasi jalan, medis (CT Scan), kendaraan otonom, dan industri.

Model untuk Object Segmentation

  • U-Net
  • Mask R-CNN
  • YOLOv8 Segmentation

Dataset dan Anotasi

  • Dataset dengan anotasi mask
  • Format polygon atau pixel mask
  • Tools anotasi: CVAT, LabelMe

Contoh Implementasi YOLOv8 Segmentation

pip install ultralytics
    
from ultralytics import YOLO

model = YOLO("yolov8n-seg.pt")

model.train(
    data="segmentation_dataset.yaml",
    epochs=50,
    imgsz=640
)

results = model("image.jpg", show=True)

Manfaat Object Segmentation

  • Akurasi deteksi objek lebih detail
  • Digunakan pada sistem vision tingkat lanjut
  • Mendukung project AI skala industri
