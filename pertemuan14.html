Multi-Label Object Detection – Deteksi APD

Pengenalan Multi-Label Detection

Multi-Label Object Detection adalah teknik dalam Computer Vision di mana satu gambar dapat memiliki lebih dari satu objek dan setiap objek dapat diklasifikasikan ke dalam beberapa label secara bersamaan.

Studi Kasus: Deteksi APD

Deteksi APD (Alat Pelindung Diri) digunakan untuk memantau keselamatan kerja di area industri. Contoh APD yang dideteksi meliputi helm, rompi, masker, dan sepatu safety.

Contoh Label APD

  • Helmet
  • Vest
  • Mask
  • Safety Shoes
  • Gloves

Dataset dan Anotasi

  • Gambar pekerja di area kerja
  • Anotasi bounding box multi-label
  • Tools anotasi: LabelImg / Roboflow

Model untuk Multi-Label Detection

  • YOLOv8
  • YOLOv5
  • Faster R-CNN

Alur Kerja Deteksi APD

  1. Pengumpulan dataset APD
  2. Anotasi multi-label
  3. Training model object detection
  4. Evaluasi hasil deteksi
  5. Implementasi real-time (kamera)

Contoh Implementasi YOLOv8

pip install ultralytics
    
from ultralytics import YOLO

model = YOLO("yolov8n.pt")

Manfaat Deteksi APD

  • Meningkatkan keselamatan kerja
