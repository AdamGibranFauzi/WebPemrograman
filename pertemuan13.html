Object Detection – Deteksi Jalan Berlubang

Pengenalan Object Detection

Object Detection adalah salah satu bidang dalam Computer Vision yang bertujuan untuk mendeteksi dan mengklasifikasikan objek dalam sebuah gambar atau video beserta lokasi objek tersebut.

Studi Kasus: Deteksi Jalan Berlubang

Deteksi jalan berlubang bertujuan untuk membantu pemerintah dan pengguna jalan dalam mengidentifikasi kerusakan jalan secara otomatis menggunakan citra atau video.

Dataset yang Digunakan

  • Gambar jalan berlubang dan tidak berlubang
  • Format anotasi bounding box
  • Dapat menggunakan LabelImg untuk pelabelan data

Algoritma Object Detection

  • YOLO (You Only Look Once)
  • SSD (Single Shot Detector)
  • Faster R-CNN

Alur Kerja Deteksi Jalan Berlubang

  1. Pengumpulan data citra jalan
  2. Pelabelan data (bounding box)
  3. Training model object detection
  4. Evaluasi model
  5. Implementasi pada gambar atau video

Contoh Implementasi YOLOv8

pip install ultralytics
    
from ultralytics import YOLO

# Load model
model = YOLO("yolov8n.pt")

# Train model
model.train(
    data="dataset.yaml",
    epochs=50,
    imgsz=640
)

# Prediksi gambar
results = model("jalan_berlubang.jpg", show=True)

Manfaat Deteksi Jalan Berlubang

  • Meningkatkan keselamatan pengguna jalan
  • Mempercepat proses perbaikan jalan
  • Mendukung sistem smart city
